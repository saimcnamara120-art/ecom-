eCommerce Marketplace

Welcome to my eCommerce Marketplace project developed using Django, MariaDB and Django REST Framework.

Project Features

  • User registration and login
  • Buyer and Vendor accounts
  • Store management
  • Product management
  • Shopping cart
  • Checkout and order management
  • Product reviews
  • RESTful API for products and stores
  • MariaDB database backend
